# VOTEX — Homepage

Simple static homepage scaffold (HTML, CSS, JS).

Files:
- `home.html` — main page
- `votex.css` — styles
- `votex.js` — small interactions (safe checks for optional elements)

Open `home.html` in your browser to view the page. If you want a local static server, from this folder run:

```bash
# Python 3
python -m http.server 8000
```

Then open http://localhost:8000/home.html
